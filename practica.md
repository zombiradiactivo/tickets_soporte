## 📝 Ejercicios para los Alumnos

### 1. Refactorización (Obligatorio)
- [ ] **Separar el código en módulos** dentro de la carpeta `src/`.
- [ ] **Crear clases** para cada componente del pipeline.
- [ ] **Implementar inyección de dependencias** para mejorar la modularidad.

---

### 2. Pipeline (Obligatorio)
- [ ] **Crear una clase `Pipeline`** que encadene todos los pasos del proceso.
- [ ] **Implementar logging** detallado en cada etapa.
- [ ] **Añadir manejo de errores robusto** (try/except específicos).

---

### 3. Tests (Obligatorio)
- [ ] **Crear tests unitarios** para cada módulo individual.
- [ ] **Implementar tests de integración** para validar el pipeline completo.
- [ ] **Alcanzar al menos un 80% de cobertura** de código.

---
